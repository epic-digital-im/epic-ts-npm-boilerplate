# Getting started

## Installation

> Clone this repository: 
```bash 
git clone https://github.com/epicdigitalmedia/epic-ts-npm-boilerplate
```
### Open the directory and run the script line:

```bash
cd epic-ts-npm-boilerplate 
```
```bash
yarn # or npm i
```
